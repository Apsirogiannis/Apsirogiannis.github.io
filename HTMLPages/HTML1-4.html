Commmon HTML Tags

These are the most commonly used HTML tags and a short description of each

Block Elements

Block elements take up their own line and take up the full width of the page.

body

defines the document's body

div

defines a section in a document

main

Specifies the main content of a document

footer

Defines a footer for a document or section

li

Defines a list item

head

defines information about the document

section

Defines a section in a document

article

Defines an article

p

Defines a paragraph

html

This tag Defines an HTML document

header

Defines a header for a document or section

nav

This tag defines navigation links

aside

Defines content aside from the page content

Inline Elements

Inline elements only takee up as much space on a line as they need to. They do not start new lines.

a

this tag defines a hyperlink

strong

Bolds important text

em

Defines emphasized text

img

This tag defines an image

span

This tag defines a section in a document
